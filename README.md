# Countdown Timer

## Overview
A simple, user-friendly countdown timer application built with Python Tkinter. Users can set hours, minutes, and seconds to start a countdown that displays in real-time until zero.

## Features
- Set custom hours, minutes, and seconds
- Real-time countdown display (hh:mm:ss format)
- Input validation (numbers only)
- Pop-up alert message when timer finishes
- Clean, modern GUI design with custom colors

## Technologies Used
- **Language:** Python
- **Framework:** Tkinter (GUI)

## How It Works
1. User enters hours, minutes, and seconds
2. Click Start Timer button
3. Timer counts down every second
4. Pop-up message appears when time reaches zero
